# 데이터베이스 클론 스크립트 사용 가이드

이 폴더에는 데이터베이스를 완전히 재생성하는 SQL 스크립트들이 포함되어 있습니다.

## 파일 구성

1. **dbclone_01_drop_and_schema.sql**
   - 기존 테이블 및 타입 삭제
   - ENUM 타입 생성
   - 모든 테이블 생성
   - 인덱스 생성
   - 코멘트 추가

2. **dbclone_02_exercises.sql**
   - 기본 운동 목록 시드 데이터 (3대 운동 + 대체 운동)
   - Free Exercise DB 데이터 (4000+ 운동)

3. **dbclone_03_bench_press_standards.sql**
   - 벤치프레스 표준 데이터 (남성 + 여성)
   - 체중별 및 나이별 데이터 포함

4. **dbclone_04_squat_standards.sql**
   - 스쿼트 표준 데이터 (남성 + 여성)
   - 체중별 및 나이별 데이터 포함

5. **dbclone_05_deadlift_standards.sql**
   - 데드리프트 표준 데이터 (남성 + 여성)
   - 체중별 및 나이별 데이터 포함

## DBeaver에서 실행 방법

### 순서대로 실행

1. **dbclone_01_drop_and_schema.sql** 실행
   - 모든 기존 데이터가 삭제됩니다!
   - 스키마가 생성됩니다.

2. **dbclone_02_exercises.sql** 실행
   - 운동 목록이 삽입됩니다.

3. **dbclone_03_bench_press_standards.sql** 실행
   - 벤치프레스 표준 데이터가 삽입됩니다.

4. **dbclone_04_squat_standards.sql** 실행
   - 스쿼트 표준 데이터가 삽입됩니다.

5. **dbclone_05_deadlift_standards.sql** 실행
   - 데드리프트 표준 데이터가 삽입됩니다.
   - 완료!

## 주의사항

- ⚠️ **모든 기존 데이터가 삭제됩니다!**
- 각 파일은 순서대로 실행해야 합니다.
- 각 파일 실행 후 완료 메시지가 표시됩니다.
- 파일이 큰 경우 DBeaver에서 실행 시간이 걸릴 수 있습니다.

## 포함된 데이터

- ✅ 모든 테이블 스키마
- ✅ 기본 운동 목록 (3대 운동 + 대체 운동)
- ✅ Free Exercise DB 데이터 (4000+ 운동)
- ✅ 벤치프레스 표준 데이터 (남성/여성, 체중별/나이별)
- ✅ 스쿼트 표준 데이터 (남성/여성, 체중별/나이별)
- ✅ 데드리프트 표준 데이터 (남성/여성, 체중별/나이별)

## 문제 해결

- 파일 실행 중 오류가 발생하면, 이전 파일들이 정상적으로 실행되었는지 확인하세요.
- `ON CONFLICT DO NOTHING` 구문이 사용되어 있어 중복 실행 시에도 안전합니다.
- 각 파일은 독립적으로 실행 가능하도록 설계되었습니다.
