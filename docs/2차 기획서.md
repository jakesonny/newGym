## 시스템 요약

- 회원 체력 평가, 운동 기록, 목표 관리 등 통합 관리
- 트레이너 업무 효율화 및 회원 피드백 제공

### 사용자 역할
- **관리자**: 트레이너 승인, 전체 회원/트레이너 관리
- **트레이너**: 회원 관리, 평가/운동 기록 입력
- **회원**: 본인 정보, 운동 기록 조회

---

## 핵심 구현 기능

- 회원/PT/회원권/부상/운동 기록/운동 루틴 관리
- 자동 점수 및 Strength Level 계산, 볼륨·1RM 분석, 운동 캘린더 제공
- 목표 설정/진행률, 트레이너 코멘트
- 대시보드, 위험 신호 회원 알림
- 역할별 권한, 이메일/카카오 로그인

---

## 평가 시스템 요약

- **평가**: 초기 1회 + 정기 반복 / 6개 영역(하체근력, 심폐, 근지구력, 유연성, 체성분, 부상)
- **점수**: 영역별 0~100점, 가중 평균(부상/심폐/근지구력 20%, 하체/체성분 15%, 유연성 10%)
- **시각화**: 레이더, 라인, 비교 차트
- **부상부위**는 평가에서 자동 제외

## 운동 기록 및 Strength Level 시스템

### Strength Level이란?

회원의 운동 기록(무게, 횟수)과 체중, 성별, 나이를 기반으로 **StrengthLevel.com의 일반 헬스 이용자 평균 데이터**에 따라 운동별 상대적 강도를 평가하는 시스템입니다.

### 평가 대상 및 기준

- **현재 평가 가능**: 빅3 운동 (벤치프레스, 스쿼트, 데드리프트)
- **평가 기준**:
  - 체중별 기준 (우선): 남성 50~140kg, 여성 40~120kg
  - 나이별 기준: 15~90세 (체중별 기준 없을 시)

### Strength Level 5단계

| 레벨 | 한글명 | 색상 |
|------|--------|------|
| BEGINNER | 초급 | 회색 |
| NOVICE | 입문자 | 파란색 |
| INTERMEDIATE | 중급자 | 초록색 |
| ADVANCED | 고급자 | 주황색 |
| ELITE | 엘리트 | 금색/빨간색 |

*등급별 한글명은 피드백 부탁드립니다.*

### 자동 계산 기능

- **1RM 계산**: 최대 1회 반복 가능한 무게 추정 (Epley 공식)
- **상대적 강도**: 체중 대비 무게 비율
- **Strength Level 판정**: 1RM, 체중, 성별, 나이 기반 자동 판정
  - StrengthLevel.com의 일반 헬스 이용자 평균 데이터 사용

### 활용 방법

- 개인 회원도 앱을 통해 자신의 Strength Level을 직접 확인하고 목표를 설정할 수 있습니다.
- 회원 동기 부여: 현재 레벨 확인 및 목표 레벨 설정
- 트레이너 평가: 객관적인 레벨로 회원 능력 평가
- 진행 추적: 시간에 따른 레벨 변화 그래프로 확인

## 앞으로 추가될 기능

### Strength Level 시각화

- Strength Level 표시: 운동 기록 화면 및 대시보드에 현재 레벨, 1RM, 진행률 표시
- 1RM 트렌드 그래프: 시간에 따른 1RM 변화를 선 그래프로 표시

### 운동 분석 강화

- 볼륨 트렌드 그래프: 시간에 따른 총 볼륨 변화 추이
- 대시보드 Strength Level 카드: 빅3 운동 Strength Level 한눈에 표시

### 스마트 기능 추가

- 운동 루틴 무게 자동 제안: Strength Level 기반 권장 무게 제안
- 운동 목록 최근 수행 우선순위: 최근 수행한 운동 상단 표시

### 추가 기능

- 운동 일지: 성장 추이를 우상향 그래프로 표시
- 1RM 추정 모달: 빅3 운동 예상 1RM 종합 표시
- 운동 루틴 자동 생성: 자주 하는 운동을 루틴으로 묶어주기
- 운동 한글 이름 등록: 870개 운동의 한글 이름 추가 (운동 선택 시 편의성 향상)

## 시스템 활용 가이드

### 트레이너 활용 가이드

**회원 등록 시**
1. 회원 기본 정보 입력
2. 회원권 등록, PT 횟수 설정
3. 초기 평가 수행 (6개 영역 평가 항목 입력)
4. 회원 목표 설정

**정기 평가 시**
1. 정기 평가 생성 (예: 월 1회)
2. 각 영역별 평가 항목 입력
3. 레이더 차트로 변화 추이 확인

**PT 세션 진행 시**
1. PT 세션 등록
2. 운동 기록 입력 (Strength Level 자동 계산)
3. 운동 루틴 생성 및 추천

**회원 관리 시**
- 회원 대시보드 확인 (목표, 진행률, 운동 캘린더, 운동 기록 분석)
- 위험 신호 회원 확인
- 맞춤형 운동 프로그램 제안

### 회원 활용 가이드

- 본인 정보 확인: 목표, 진행률, 운동 일정, 능력치 레이더 차트
- 운동 기록 확인: 과거 운동, 부위별 볼륨 분석, Strength Level
- 목표 달성 추적: 목표 및 진행률, 트레이너 코멘트 확인

## 주요 특징

- **자동화된 계산**: 모든 점수와 레벨 자동 계산, 측정값만 입력
- **시각적 표현**: 레이더 차트, 라인 차트, 진행 바로 직관적 표현
- **시간에 따른 추적**: 초기 평가 기준으로 변화 추이 추적
- **안전한 평가**: 부상 영역 자동 제외
- **일반 헬스 이용자 평균 기준**: Strength Level은 StrengthLevel.com의 일반 헬스 이용자 평균 데이터 사용
